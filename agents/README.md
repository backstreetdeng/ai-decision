## 智能体定义

### 1. market_strategy_agent - 市场战略分析师

| 属性 | 值 |
|------|-----|
| **角色定义** | 5年汽车行业市场分析师，精通市场分析、竞品研究、政策解读 |
| **核心能力** | 乘用车市场大盘数据分析、竞品格局与份额分析、政策影响评估、市场机会识别 |
| **输出** | 市场宏观态势报告、战略性机会清单、风险预警提示 |
| **约束边界** | 不输出具体定价建议、不做营销策略建议 |

### 2. user_insight_agent - 用户需求洞察分析师

| 属性 | 值 |
|------|-----|
| **角色定义** | 专注用户研究的数据分析师，擅长用户画像、需求挖掘、配置偏好分析 |
| **核心能力** | 目标用户圈定与画像生成、用户偏好深度分析、车型配置优先级排序、购车决策因子分析 |
| **输出** | 标准用户画像、配置需求优先级列表、购车决策因子排序 |
| **约束边界** | 仅输出需求洞察，不做产品定义、不输出营销策略 |

## 双智能体协同流程

```
1. 用户输入问题 → 意图识别
2. 判断是否需要双Agent协同
3. market_strategy_agent 先执行（背景信息）
4. user_insight_agent 后执行（具体画像）
5. 知识库检索提供数据支撑
6. 结果汇总生成完整报告
```

## 使用示例

### Python 调用示例

```python
from agents import AgentInvoker, AgentType

# 初始化
invoker = AgentInvoker(llm_gateway, knowledge_base)

# 单Agent调用
result = invoker.invoke_agent(
    AgentType.MARKET_STRATEGY,
    "30万插混SUV市场机会分析"
)

# 双Agent协同调用
dual_results = invoker.invoke_dual_agents(
    query="30万纯电SUV市场机会分析"
)
```

### CLI 调用示例

```bash
# 市场分析
python -m agents invoke market_strategy_agent "30万插混SUV市场机会分析"

# 用户洞察
python -m agents invoke user_insight_agent "30万纯电SUV用户画像"

# 双Agent协同
python -m agents invoke dual "30万纯电SUV市场机会分析"
```

## 数据来源

| 数据类型 | 来源 | 存储 |
|---------|------|------|
| 市场数据 | 乘联会、中汽协、厂商公告 | PostgreSQL |
| 竞品数据 | 汽车之家、懂车帝 | PostgreSQL |
| 用户调研 | 内部调研、公开报告 | pgvector |
| 口碑舆情 | 社交媒体、论坛 | pgvector |
| 政策文件 | 政府网站、公告 | pgvector |

## 技术栈

- **LLM**: Qwen2.5-7B-Instruct (QLoRA微调)
- **向量模型**: BGE-large-zh-v1.5
- **向量数据库**: pgvector
- **框架**: LangChain / 自研轻量引擎
